# RAXUS — IT Operations Platform

> 🚀 **Deploy on Render in 2 clicks** | Observabilité · Automatisation · Sécurité

RAXUS est une plateforme unifiée de gestion des opérations IT inspirée des meilleurs outils du marché.

---

## 🌐 Quick Deploy on Render

### 1. Connect GitHub to Render
- Go to [render.com](https://render.com)
- Connect your GitHub account

### 2. Deploy with Blueprint
- Click **"New Blueprint"**
- Select `filelien/rsw` repository
- Render will detect `render-simple.yaml`
- Click **"Deploy"** 🚀

### 3. Access your app
- **Frontend**: `https://raxus.onrender.com`
- **API**: `https://raxus-api.onrender.com`
- **API Docs**: `https://raxus-api.onrender.com/docs`
